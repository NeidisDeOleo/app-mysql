🐳 App Web PHP + MySQL con Docker
📌 Descripción

Aplicación web básica en PHP que se conecta a una base de datos MySQL usando Docker Compose.

🚀 Ejecución
git clone https://github.com/NeidisDeOleo/app-mysql.git
cd app-mysql
docker compose up -d --build

Abrir en el navegador:
http://localhost:8080
